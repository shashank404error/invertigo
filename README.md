## About The Project

Invertigo supercharges your reading experience. Say goodbye to slow, straining reading! Invertigo analyzes and fine-tunes the font on any webpage, customizing it for optimal speed and focus. This clever technology ensures you devour information like never before, while improving comprehension and reducing eye fatigue. Download Invertigo today and unlock the power of effortless, efficient reading!

## Getting Started

- Clone the repo
- The popup.html and popup.js controls the extension design
- The content.js does the actual job
- The background.js binds everything together.
- Permissions are maintained in manifest.json

### Prerequisites

- Should have chrome web browser installed locally

### Installation

1. Go to chrome://extensions/
2. At the top right, turn on Developer mode
3. Click Load unpacked.
4. Find and select the app or extension folder.
